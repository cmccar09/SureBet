1 · System Overview

Each horse in each race is scored on a 0 → ~200 scale (practical range 30–150) by summing weighted signals across 16 sections. The highest scorer per race is selected as the pick. Scores are then banded into 4 confidence tiers that drive stake sizing. The pipeline runs automatically every 15 minutes via EventBridge → Lambda.


Betfair API → runner data going_conditions.json → going scores DynamoDB SureBetBets → history form_enricher → Racing Post per-run history comprehensive_pick_logic.py → score 4-tier grade → stake lambda_api_picks.py → UI + auto-record

🟢 EXCELLENT · 70+ pts · 2.0× stake
🟡 GOOD · 55–69 pts · 1.5× stake
🟠 FAIR · 40–54 pts · 1.0× stake
🔴 POOR · <40 pts · 0.5× stake

Class 5/6 races have a hard ceiling of 80 pts regardless of accumulated score. This prevents form-streak inflation from creating false certainty in the lowest, most random race class.

§5 Going Conditions KEY SIGNAL

Going is the single most critical discriminator in UK/Irish racing. Base weight = 16 pts, doubled to 32 pts in extreme going (Heavy/Soft or very firm).

§5a Heavy Going Penalty: ALL horses lose −12 pts when going = Heavy (not just Soft). Lesson: El Gavilan 100pts, 3/1 fav → 5th. Heavy ground fundamentally randomises results.
§5b Graded Race Correction: If a going penalty is applied but the race is Grade 1/2/3 or Listed, the penalty is removed. Elite horses are prepared for all conditions. Lesson: James Du Berlais (Grd2 Navan, form 55-1632) won despite penalty being triggered.
§5c C/D Marker Bonus: CD = +18pts (or +26 in graded race); C only = +9pts; D only = +9pts. These are hard proof of ability — the form string alone can't capture this.
§6 Database History

Queries DynamoDB SureBetBets table for past wins by this horse name. Any historical win = +15 pts flat.

Uses get_track_insights(course) which analyses outcomes from earlier races today at the same course. If a dominant pattern is found (e.g. "form-backed favourites winning"), horses that match that pattern get up to +8 pts (capped).

7 · Special Case Rules

Graded Race (Grd1/2/3 / Listed)

Going penalties are removed for graded races. Elite horses are prepared for all conditions; penalising based on 4–6 form digits is a false negative. CD bonus also gets +8 bonus on top of base.

Cheltenham Festival (10–13 Mar)

Full Cheltenham-specific overlay applies: trainer history (Mullins 32% SR), Irish bias (73% of winners), form patterns ('1111' = +20), jockey records, race specialties.

Class 5/6 Hard Cap

Maximum score of 80 pts enforced after all sections. Prevents form-streaks overwhelming the unpredictability signals in lowest-grade fields.

Dynamic Weights Override

The continuous learning system (get_dynamic_weights()) loads from DynamoDB / a weights JSON and may adjust DEFAULT_WEIGHTS based on recent results. Falls back to defaults if unavailable.
